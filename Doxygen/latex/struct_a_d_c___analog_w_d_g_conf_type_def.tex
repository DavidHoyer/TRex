Configure the ADC analog watchdog mode\+: single/all/none channels. For Analog Watchdog 1\+: Configure the ADC analog watchdog mode\+: single channel or all channels, ADC groups regular and-\/or injected. For Analog Watchdog 2 and 3\+: Several channels can be monitored by applying successively the AWD init structure. Channels on ADC group regular and injected are not differentiated\+: Set value \textquotesingle{}ADC\+\_\+\+ANALOGWATCHDOG\+\_\+\+SINGLE\+\_\+xxx\textquotesingle{} to monitor 1 channel, value \textquotesingle{}ADC\+\_\+\+ANALOGWATCHDOG\+\_\+\+ALL\+\_\+xxx\textquotesingle{} to monitor all channels, \textquotesingle{}ADC\+\_\+\+ANALOGWATCHDOG\+\_\+\+NONE\textquotesingle{} to monitor no channel.
